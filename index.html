PSO Class Data Visualization

Welcome, you can use this tool to view and compare class stat spread and growth.
The data used in this tool is scraped from PSO-World.com, and converted into json format.
If the stats are not correct, blame pso-world ;).

RadarChart

The radarchart allows you to compare all the stats of multiple classes at any given level (including maxed out stats).
The scale on the radarchart is percentage based, this means that each stat is compared to the class that has the highest value in this stat at the chosen level.
For example, if we decide to compare maxed out stats, a HUmar would score approximately 85% on ATP.
Since his 1397 ATP stat is about 85% of 1639, the HUcasts maxed out ATP which is the highest in game.

LineGraph

The linegraph allows you to compare the growth of multiple stats of multiple classes in a given level range.
